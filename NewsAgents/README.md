#  ArticleCraft AI

**ArticleCraft AI** is a working **multi-agent content generation app** built with [CrewAI](https://github.com/joaomdmoura/crewai) and high-performance **LLMs**.  

It uses a team of specialized AI agents to **research**, **write**, and **proofread** articles in a **sequential workflow** all from a single topic you provide.

---

##  Purpose

The goal of ArticleCraft AI is to demonstrate how **CrewAI** can coordinate multiple AI agents to complete a complex writing workflow, while leveraging **LLMs** for high-quality, natural language output.  

This project shows how to:
- Structure **multi-agent pipelines** for real tasks.
- Use **different roles** for different stages of content creation.
- Manage **rate limits** for production reliability.

---

##  How It Works

1. **Researcher** — gathers data and identifies trends.
2. **Writer** — creates a coherent, well-structured article.
3. **Proof Reader** — ensures readability and correctness.

Agents work **sequentially** so that each step improves upon the previous one.
